# タスク管理アプリ

このプロジェクトは、Flask と PostgreSQL を使用したシンプルなタスク管理アプリです。ユーザーはアカウントを作成し、タスクの追加・完了・削除ができます。

## 特徴
- ユーザー登録・ログイン機能
- タスクの追加・完了・削除機能
- PostgreSQL を使用したデータ管理
- セッションを利用したユーザー認証

## 環境構築と実行方法

### 1. 必要なツールのインストール
このアプリを実行するには、以下のツールが必要です。
- Python 3.x
- PostgreSQL
- pip（Python パッケージ管理ツール）
